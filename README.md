# Audio → Text → AI Analyzér

Tento projekt umí:
- nahrát audio soubor (WAV / MP3)
- převést jej na text pomocí offline enginu PocketSphinx (funguje bez internetu)
- poslat text do školního AI API
- získat shrnutí a typ obsahu (řeč / píseň)
- uložit historii nahrávek do `history.json`
